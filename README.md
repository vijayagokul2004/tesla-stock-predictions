# 📈 Tesla Stock Price Movement Prediction – Machine Learning Project

## 📌 Problem Statement
Predict Tesla stock price movement using historical financial market data to understand short-term trends and patterns.

---

## ⚙️ Approach / Methodology

### 1. Data Understanding
- Collected historical Tesla stock data
- Analyzed price trends and market behavior

### 2. Data Preprocessing
- Cleaned dataset and handled missing values
- Created meaningful features for prediction
- Normalized data where required

### 3. Model Building
- Implemented Logistic Regression
- Tested Support Vector Classifier (SVC)
- Applied XGBoost for improved performance

### 4. Model Evaluation
- Accuracy
- Precision
- Recall
- F1 Score

---

## 📊 Results
- Accuracy: 54.7%
- Precision: 57.8%
- Recall: 67.0%
- F1 Score: 62.1%

- Supports financial decision-making
- Provides insights for algorithmic trading systems
- Useful for market behavior analysis
